#Add Marker with Comment
(C) あかつきみさき(みくちぃP)

##このスクリプトについて
プロンプト入力したテキストのコメント付きマーカーを追加するスクリプト.
何も選択していない時はコンポジションマーカーに,レイヤーを選択している時はすべてのレイヤーに追加されます.

##使用方法
1. ファイル→スクリプト→スクリプトファイルの実行から実行.
